Link Buttons

A link can also be styled as a button, by using CSS:

                <style>
                a:link, a:visited {
                    background-color: #f44336;
                    color: white;
                    padding: 15px 25px;
                    text-align: center;
                    text-decoration: none;
                    display: inline-block;
                  }
                  
                  a:hover, a:active {
                    background-color: red;
                  }
                </style>
